# Retail Sales Analysis using MySQL

## Project Overview
This project focuses on analyzing a retail sales dataset using MySQL. The analysis was performed on the Superstore dataset to generate business insights related to sales, profit, customers, products, and regions.

## Tools Used
- MySQL Workbench
- SQL
- Microsoft Excel (for data cleaning)

## Dataset
Superstore Retail Sales Dataset

## Objectives
- Analyze overall sales and profit
- Identify top-performing products
- Find high-value customers
- Analyze regional sales performance
- Discover loss-making products
- Study monthly sales trends
